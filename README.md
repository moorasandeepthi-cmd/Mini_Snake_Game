# 🐍 Mini Snake Game

## 📌 Description
Mini Snake Game is a simple and interactive browser-based game inspired by the classic Snake game. The player controls the snake using keyboard arrow keys to collect food, increase the score, and grow the snake's length. The game ends when the snake collides with the walls or its own body. This project demonstrates the implementation of game logic and interactive user interfaces using web technologies.

## 🎯 Features
- Classic Snake gameplay
- Keyboard controls using arrow keys
- Random food generation
- Real-time score tracking
- Snake grows longer after eating food
- Collision detection with walls and the snake's body
- Game over functionality
- Easy-to-use and responsive interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 🚀 How to Run the Project
1. Clone the repository:
   ```bash
   git clone https://github.com/moorasandeepthi-cmd/Mini_Snake_Game.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Mini_Snake_Game
   ```

3. Open the `index.html` file in your preferred web browser.

## 🎮 How to Play
- Use the **Arrow Keys** to control the snake.
- Eat the food to increase your score.
- The snake grows longer after eating food.
- Avoid colliding with the walls or the snake's own body.
- Try to achieve the highest score possible.
